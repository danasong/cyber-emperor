# cyber-emperor

一个面向 OpenClaw 的本地 skill，专门用于把复杂任务编排成“三省六部式”多 Agent 协作流程。

## 文件

- `SKILL.md`：主说明文档，可直接被 agent 阅读使用

## 核心特点

- 明确什么时候用、什么时候不用
- 把多 Agent 职责拆为三省六部
- 强制要求实际使用 `sessions_spawn` 派工
- 对复杂编码任务强制要求结合 `claude-code-hook`
- 强调输入、输出、验收、回收四个工程动作

## 适合场景

- 新功能开发
- 多模块并行改造
- 复杂文档与工程交付协作
- 需要“方案-实现-测试-文档”分工的任务

## 不适合场景

- 纯查询
- 单文件小修
- 明显拆分收益低于沟通成本的工作
